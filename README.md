webIS
=====

命名方式：
-----

#### 关键字开头：

##### 前两位：特性（标签或其他新特性）

* 00: form新特性
* 01: input新特性
* 02: video新特性
* 03: ping新特性
* 04: Body新特性
* 05: source新特性
* 06: WebRTC新标准

##### 中间两位：环境

* 00: 用户点击
* 01: 不需交互
* 02: 输入框输入

##### 后一位：漏洞类型

* 0: XSS
* 1: CSRF
* 2: DOS
* F: Auxiliary

-----

###### 注：

两位数字用十六进制表示

------

*Example: 00_00_0_form新特性￥%&**
