------
* 特性：`<progress>` 标签定义运行中的进度（进程）。可以使用` <progress> `标签来显示 JavaScript 中耗费时间的函数的进度。
* 格式：`<progress>`<br>
        `<span id="objprogress">85</span>%`<br>
        `</progress>`
* 属性：max 	number 	规定任务一共需要多少工作。<br>
       value 	number 	规定已经完成多少任务。<br>
       
------
安全性：基本无新属性方法漏洞；
        或可在对JS的读取中入手；
